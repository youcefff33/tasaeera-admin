### الصفحات الرئيسية
1. **Dashboard** - لوحة التحكم الرئيسية
2. **Users Management** - إدارة المستخدمين
3. **Stores Management** - إدارة المتاجر
4. **Products Management** - إدارة المنتجات
5. **Prices Review** - مراجعة الأسعار
6. **Reports Management** - إدارة البلاغات
7. **Analytics** - التحليلات والرسوم البيانية
8. **Settings** - الإعدادات المتقدمة
9. **Admin Panel** - لوحة إدارة المشرفين

### التقنيات
- **Framework**: React 18
- **Language**: TypeScript
- **State Management**: Redux Toolkit
- **HTTP Client**: Axios
- **Routing**: React Router v6
- **UI Framework**: Ant Design / Material-UI
- **Charts**: Recharts
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

### البدء السريع

```bash
# تثبيت الحزم
npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
npm run build

# معاينة الإنتاج
npm run preview
```
